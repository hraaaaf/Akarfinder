# PARTNER_PAGES_EXPERIENCE.md - Experience pages partenaires AkarFinder

## Objectif

AkarFinder peut proposer aux promoteurs et agences partenaires des pages dediees enrichies, uniquement si les donnees respectent le standard AkarFinder.

Ces pages ne transforment pas AkarFinder en marketplace. Elles montrent une experience partenaire autorisee, distincte des resultats web externes.

## Promoteurs

Une page promoteur peut inclure:
- identite partenaire autorisee;
- zones couvertes;
- projets actifs;
- page projet detaillee;
- tranches;
- typologies;
- plans 2D fournis par le partenaire;
- appartement temoin;
- galerie autorisee;
- localisation exploitable;
- proximite, mobilite, quartier;
- disponibilite indiquee comme donnee partenaire;
- CTA brochure, rendez-vous, demande d'information.

## Agences

Une page agence peut inclure:
- agence virtuelle;
- zones couvertes;
- specialites;
- biens structures selon le standard AkarFinder;
- contact autorise;
- WhatsApp uniquement si autorise dans une version partenaire reelle;
- demandes qualifiees;
- lecture quartier.

## Resultats partenaires vs web externe

Les resultats partenaires peuvent etre enrichis si les donnees sont autorisees.

Les resultats web externes restent limites:
- apercu limite;
- source originale visible;
- pas de contact direct;
- pas de galerie;
- pas de reutilisation d'image non autorisee.

Ordre conceptuel futur:
1. Resultat partenaire autorise tres pertinent.
2. Promoteur partenaire si intention neuf.
3. Agence premium pertinente.
4. Agence partenaire.
5. Resultat web externe limite.

Regle permanente: pertinence d'abord, partenaire ensuite, qualite de fiche ensuite, source externe en dernier.

## Demo uniquement

La demonstration actuelle:
- ne cree aucun backend;
- ne cree aucune API;
- ne stocke aucun fichier;
- ne cree aucun vrai lead;
- ne represente aucun partenaire reel;
- ne modifie pas le Search Gateway;
- ne modifie pas le ranking reel.

Tous les contenus doivent rester marques comme:
- Mode demo;
- Donnees fictives;
- Exemple non contractuel;
- A confirmer aupres du partenaire.

## Wording interdit

Ne jamais presenter ces pages avec:
- verifie;
- certifie;
- officiel;
- fiable;
- garanti;
- meilleur;
- prix reel;
- annonce verifiee;
- annonce fiable;
- partenaire officiel;
- agence de confiance;
- vente garantie;
- toutes les annonces;
- marketplace;
- exhaustif.

## Wording autorise

Wording autorise:
- Promoteur partenaire;
- Agence premium;
- Agence partenaire;
- Page partenaire autorisee;
- Apercu limite;
- Donnees fictives;
- Exemple non contractuel;
- Plan 2D fourni par le partenaire;
- Plan indicatif;
- A confirmer sur place;
- Reperes indicatifs;
- Demande qualifiee demo.
